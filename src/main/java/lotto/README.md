# 로또(자동)

## 구현할 기능 목록
1. 기본 숫자 (NumberGenerator)
    - 1~45 범위의 숫자를 가진 숫자 

2. 로또 (Lotto)
    - 로또 번호를 가진다
    - 랜덤 번호를 생성한다

3. 로또 머신 (LottoMachine)
    - 입력한 장 수 만큼 로또 생성 
                                                        
4. 상금 (Prize)
    - 일치 개수와 당첨금으로 상금 계산

5. 로또 당첨 확인기 (WinningChecker)
    - 번호별 일치 개수 확인
                                                    
6. 로또 분석기 (LottoAnalyzer)
    - 일치 개수 별 당첨 수 확인 (3개일치 로또 - 1개)
    - 수익율 계산 

7. 로또 유틸 (LottoUtil)
    - 구매 장수 계산

8. LottoController
    - 로또 어플리케이션
